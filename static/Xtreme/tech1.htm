התקנת Java

מנוע המשחק נכתב בשפת Java (גירסה 7), ולכן יש להתקין תחילה תמיכה בטכנולוגיה זו.

כדי לעשות זאת, גשו לאתר ההתקנה של חברת SUN, ולחצו על Download Now (או Manual Download). לאחר מכן, עקבו אחר ההוראות באתר עד להורדת קובץ ההתקנה למחשב, וסיום התקנת התוכנה.

כעת, מחשבכם מסוגל להריץ תוכניות Java.

התקנת מנוע המשחק

בשלב זה, תוכלו להוריד את קובץ התקנת המשחק. חבילת ההתקנה שהורדתם כוללת מספר קבצים: מנוע המשחק, קובץ batch שיאפשר לכם להריץ אותו בקלות בסביבת windows ומספר שורדים לדוגמא. מומלץ לשמור את כל קבצי המשחק בתיקיה אחת, המיועדת להם בלבד.

כדי להפעיל את המשחק, יש לגשת לספריה שיצרתם, ולהריץ את הקובץ cgx.bat. מנוע המשחק יטען אוטומטית את כל השורדים הנמצאים תחת התיקיה survivors.

כבר בשלב זה, תוכלו לצפות בהתמודדות אמיתית בין שורדי הדוגמא המצורפים למנוע. הנה דוגמא למשחק פשוט (לחצו על התמונה כדי להגדיל לגודל מלא):

מימין: רשימת השורדים (לכל שורד צבע משלו).
משמאל: זירת המשחק (מרחב הזכרון שהוקצה).
כל ריבוע מייצג תא בזכרון, ונצבע בצבע של השורד האחרון שכתב אליו.
למטה: עדכונים מזירת המשחק על התקדמותו.

מנוע המשחק עצמו הוא פרויקט קוד פתוח בשם corewars8086, שפותח במיוחד עבור התחרות. המשמעות היא שביכולתכם להוריד את קוד המקור של המנוע (כאמור, ב-Java), ולעשות בו כרצונכם.

השלב הבא - בניית השורדים הנוספים והרצתם!
